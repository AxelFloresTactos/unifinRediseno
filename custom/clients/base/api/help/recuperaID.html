Overview

Realiza una consulta en base de datos sobre la tabla unifin_folios y actualiza las columnas "valor", "usuario" y "fecha".

Summary

Esta petición se encarga de actualizar el número de folio/ID dependiendo el tipo de ID requerido, ya sea Backlog, Cliente, Prospecto o Solicitud. Para ello el número para identificar cada tipo de ID es:

1 = Folio de Backlog
2 = ID Cliente
3 = ID Prospecto
4 = ID Solicitud

Dichos números son el ID en la tabla unifin_folios, el cual actualizará y guardará el número de folio/ID de la tabla. Asimismo se actualizan datos de las columnas date_modified y modified_user_id con la fecha actual de cuando se realiza el cambio y con el id del usuario logueado.

Request Example


    .../recuperaID/1

Response

La respuesta mostrará un estado, una descripcion y un id para informar sobre el resultado de la petcición del servicio.
Name Type Description
Estado Notificacion Informa sobre la respuesta del servicio.
Descripcion Texto Mensaje descriptivo del servicio, informa que ha sucedido en durante el servicio.
Id Texto Mensaje descriptivo donde muestra el valor ya actualizado.

Respuesta Exitosa


{
    "Estado": "¡Exito!",
    "Descripcion": "El ID se ha generado.",
    "Id": "1002"
}

Respuesta Fallida


{
    "Estado": "¡Error!",
    "Descripcion": "El tipo de folio no existe, favor de intentar de nuevo."
}
